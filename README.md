#  TutorConnect – Instant Help For Better Learning 

TutorConnect is an on-demand EdTech platform that connects students with tutors instantly — just like booking a ride. It enables real-time doubt solving through chat, voice, or video sessions, making learning faster, personalized, and more accessible.

**Demo Video Link:** https://youtu.be/3YtcgbNR0rI
##  Project Overview

Students often get stuck while studying and lack immediate help. Existing platforms provide general solutions but not instant, personalized guidance.

TutorConnect solves this by allowing:
- Students to post doubts instantly
- Tutors to accept requests in real-time
- Live sessions for direct interaction


##  Key Features

###  Student Features
- Post doubts (text + image upload)
- Browse tutors with search & filters
- View tutor profiles and demo videos
- Book instant sessions
- Track session history (completed, pending)
- Dashboard with learning stats (sessions, hours, ratings)

###  Teacher Features
- Manage profile and teaching details
- Upload demo videos (subject, title, preview)
- Accept/reject student bookings
- View bookings (upcoming, completed)
- Earnings dashboard:
  - Total earnings
  - Monthly earnings
  - Pending payments
  - Avg earning per session
  - Earnings trend (visual stats)


##  Dashboards

###  Student Dashboard
- Overview: sessions, doubts, learning hours, ratings

- Recent doubts with status (pending/accepted)

##  System Workflow

1. Student posts a doubt (text/image)
2. System connects with available tutors
3. Tutor accepts the request
4. Live session starts (chat/voice/video)
5. Session completes → payment + rating
